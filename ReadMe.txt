(3) How to Deploy the Project:
	1. Download and Install Visual Studio.
	2. Run Visual Studio.
	3. Make sure to install Microsoft Sql Server Management Studio and import database backup file.
	4. Open Folder provided by the name Source Code in Visual Studio.
	5. Install all the libraries and dependencies which Visual Studio is recommending in order to run the project.
	6. Your Internet Connectivity should be stable in order to get the UI CDN Links.
	7. Change the Connection String in Web Config file.
